気象庁データをアプリケーション開発に利用いただけるように提供しております

気象庁で公開されている気象データをアプリケーションで利用しやすいように変換して提供するAPIサービスです。アプリケーション開発でご利用いただけます。当社でデータをホストしておりますので気象庁に迷惑をかけず運用されています。

アプリ画面はシンプルに、必要な情報を端的にまとめます

天気が気になるときに、すぐにサクッと見ることができます。必要な情報をコンパクトに表示します。

気象庁の利用規約上の問題はありませんか？

はい、気象庁 利用規約 に基づいて編集・加工した上で公開している API です。

この API に気象業務法上の問題はありませんか？

気象庁 Web サイト によると、「気象庁以外の事業者が天気や波浪等の予報の業務を行おうとする場合は、気象業務法第17条の規定により、気象庁長官の許可を受けなければならない。」と定められています。ただし、気象庁などが観測した数値や予報データを 2 次的に配信することは「予報の業務」にあたりません。
